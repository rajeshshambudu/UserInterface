What you can change

Change the URL in .hero::before and the inline background-image on .hero-title. You can also adjust the font-size, letter-spacing, or add more overlay layers.

Experiment

This implementation uses CSS-only techniques (no JS). On modern desktop browsers the parallax look is created by background-attachment: fixed. On phones, that property is often disabled for performance, so the CSS includes a graceful fallback.
